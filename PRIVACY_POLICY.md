# Privacy Policy for JobNchill Autofill

**Last Updated:** January 9, 2026

## Overview
JobNchill Autofill ("Extension") is designed to help users automate job application processes. This Privacy Policy explains what data we collect, how we use it, and your rights.

## Data Collection & Usage

### What Data We Collect
- **Form Data**: Job application form field values that you authorize to be auto-filled
- **User Preferences**: Your extension settings and preferences
- **Resume Information**: Only if you explicitly upload it for autofill purposes
- **Browser Storage**: We use Chrome's `storage` API to save your settings locally

### What We Do NOT Collect
- Personal identification data (name, email, phone) unless you enter it in forms
- Browsing history or visited websites
- Credentials or passwords
- Tracking or analytics data
- IP addresses or device identifiers

### How We Use Data
- To fill job application forms with your information
- To remember your preferences and settings
- To improve the extension functionality

## Data Storage & Security
- All data is stored **locally on your device** using Chrome's `storage` API
- No data is sent to external servers
- Data persists only in your browser profile
- Deleting the extension removes all stored data

## Third-Party Communication
This extension communicates with:
- **LinkedIn.com** - When you're actively on LinkedIn
- **Naukri.com** - When you're actively on Naukri
- **Indeed.com** - When you're actively on Indeed

These communications are only to fill forms on these websites when you use the extension.

## User Rights
- You can disable the extension anytime
- You can clear stored data by uninstalling the extension
- You have full control over what information is filled in forms

## Contact
